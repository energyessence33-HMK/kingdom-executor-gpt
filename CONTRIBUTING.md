# Contributing to Kingdom Executor GPT

## 📋 Guidelines

1. **Agent Development**
   - Follow agent template structure
   - Include comprehensive prompts
   - Test action integrations
   - Document behavior expectations

2. **Prompt Engineering**
   - Test with multiple examples
   - Include few-shot examples
   - Document expected outputs
   - Version control prompts

3. **Actions**
   - Implement error handling
   - Include retry logic
   - Document parameters
   - Test with mock data

4. **Configuration**
   - Document all parameters
   - Include example configs
   - Test in staging first
   - Monitor performance

## 🧪 Testing Requirements

- Unit tests for actions
- Prompt validation tests
- End-to-end agent tests
- Load testing for production agents

## 📊 Performance Metrics

Monitor:
- Response latency
- Token usage
- Error rates
- Action success rates

## 🔄 Deployment

1. Test in development
2. Validate in staging
3. Monitor production deployment
4. Have rollback plan ready
